# Rule: Secops Quality Gates

Define core guidelines, constraints, and behavioral principles that the Antigravity agent MUST adhere to at all times when working on this domain.

## Mandatory Constraints
- **Scope Verification**: Always verify that operations are strictly within the target scope.
- **Safety First**: Never execute operations that might delete or overwrite historical logs or uncommitted project resources without checking or prompting first.
- **Technology Alignment**: Ensure all generated components or resources strictly follow the project's preferred technologies (e.g., Python, streamlit).

## Style and Conventions
- Maintain a clean, professional, and emoji-free communication style in outputs.
- Keep diagnostic or report formats uniform and structured using tables and code blocks where appropriate.
